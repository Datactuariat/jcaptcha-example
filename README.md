jcaptcha-example
================

Quick Start
-----------

    git clone https://github.com/Datactuariat/jcaptcha-example.git
    cd jcaptcha-example
    mvn jetty:run

Requires
--------

- <abbr title="Java Development Kit">JDK</abbr> 7+
- [Apache Maven][3] 3+
